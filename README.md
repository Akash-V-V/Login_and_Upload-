This project is a responsive web application featuring a modern login page and a file upload page enhanced with smooth animations and interactive Spider-Man themed visuals. It was built to demonstrate frontend development skills using HTML, CSS, and JavaScript, with a focus on user interface design, transitions, and creative animation effects. The application highlights clean layout structuring, responsive styling, and engaging visual elements while maintaining a simple and intuitive user experience.
